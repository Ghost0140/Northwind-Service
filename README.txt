📦 Sistema de Mantenimiento de Datos - Northwind
📖 Descripción

Este proyecto consiste en el desarrollo de una aplicación web orientada a la gestión y mantenimiento de información basada en la base de datos Northwind. El sistema permite administrar entidades como productos, categorías y proveedores, facilitando las operaciones de registro, consulta, actualización y eliminación de datos (CRUD).

La solución está diseñada para mejorar la organización de la información, reducir errores en la manipulación de datos y optimizar los tiempos de respuesta en las consultas.

⚙️ Tecnologías utilizadas
ASP.NET Core MVC
Web API REST
gRPC
SQL Server
Entity Framework Core
🚀 Ejecución del proyecto

Para ejecutar el proyecto de manera local, seguir los siguientes pasos:

Clonar el repositorio:

git clone https://github.com/tu-usuario/tu-repositorio.git
Abrir la solución en Visual Studio

Configurar la cadena de conexión en el archivo:

appsettings.json
Ejecutar la base de datos (script incluido en el repositorio)
Ejecutar el proyecto
